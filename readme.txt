DMaps creates a custom metabox in post types including custom post types. Also you can create maps via [dmaps] shortcode.

== Description ==

Use DMaps via shortcode [dmaps address="the address"] everywhere in your site. Also there is the possibility to show a metabox in post types (you can select the post types in settings). The Radius field is specially usefull for Real State sites where you want to show an area but not the exact location of the property.

<a href="http://wp.dinamiko.com/demos/dmaps" target="_blank">DMaps Demo</a>

= Features =

* Responsive maps

== Installation ==

Installing "DMaps" can be done either by searching for "DMaps" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress
